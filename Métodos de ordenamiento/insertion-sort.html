Insertion Sort

¿Qué es el Insertion Sort?

Insertion Sort (Ordenamiento por inserción) es un algoritmo de ordenamiento simple y comparativo que construye el arreglo ordenado uno a uno, insertando cada elemento en su posición correcta respecto a los elementos anteriores. Es eficiente para listas pequeñas y parcialmente ordenadas, ya que su desempeño mejora cuando los datos están casi ordenados. Es similar a cómo una persona ordena cartas en la mano.

¿Cómo funciona Insertion Sort?

A medida que se avanza en la lista, los elementos anteriores se consideran ya ordenados. El algoritmo selecciona el siguiente elemento y lo desplaza hacia la izquierda hasta encontrar su posición correcta. Este enfoque permite que cada paso mejore progresivamente la organización de la lista.

Pasos del Algoritmo (Detallado)

  1. Iterar desde el segundo elemento hasta el último.
  2. Comparar el elemento actual con los elementos anteriores.
  3. Desplazar los elementos mayores hacia la derecha.
  4. Insertar el elemento actual en su posición correcta.

Complejidad

Caso Comparaciones e intercambios Complejidad
Peor caso Lista invertida O(n²)
Mejor caso Lista ya ordenada O(n)
Promedio Lista aleatoria O(n²)

El algoritmo realiza en el peor de los casos aproximadamente n²/2 comparaciones e intercambios, mientras que en el mejor de los casos (lista ordenada), solo se realizan n - 1 comparaciones y ningún intercambio.

Ventajas

  • Fácil de implementar.
  • Eficiente para listas pequeñas o parcialmente ordenadas.
  • No requiere memoria adicional.
  • Funciona bien cuando los elementos llegan de forma secuencial (como entradas en tiempo real).

Desventajas

  • Muy ineficiente en listas grandes.
  • Existen algoritmos más rápidos como Merge Sort o Quick Sort.
  • El número de comparaciones e intercambios crece rápidamente con el tamaño de la lista.
Animación: Cómo funciona el Insertion Sort
Ejemplo en PHP

function insertionSort($arr) {
    for ($i = 1; $i < count($arr); $i++) {
        $key = $arr[$i];
        $j = $i - 1;
        while ($j >= 0 && $arr[$j] > $key) {
            $arr[$j + 1] = $arr[$j];
            $j--;
        }
        $arr[$j + 1] = $key;
    }
    return $arr;
}

function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let key = arr[i];
        let j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
    return arr;
}
Ejemplo en Python

def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
