Heap Sort

¿Qué es el Heap Sort?

Heap Sort (Ordenamiento por montículos) es un algoritmo de ordenamiento basado en un árbol binario llamado heap. Utiliza la propiedad del heap para construir un árbol donde el nodo raíz es el mayor (max-heap) o el menor (min-heap), y luego intercambia la raíz con el último elemento, reduciendo el tamaño del heap y repitiendo el proceso.

¿Cómo funciona Heap Sort?

  1. Construir un max-heap a partir del arreglo.
  2. Intercambiar el elemento raíz (el mayor) con el último elemento del heap.
  3. Reducir el tamaño del heap y aplicar heapify para restaurar la propiedad del heap.
  4. Repetir hasta que el heap tenga un solo elemento.

Pasos del Algoritmo (Detallado)

  1. Construir un max-heap desde el arreglo.
  2. Intercambiar el primer elemento (máximo) con el último elemento del arreglo.
  3. Reducir el tamaño del heap y aplicar heapify al subárbol afectado.
  4. Repetir hasta que el arreglo esté completamente ordenado.

Complejidad

Caso Comparaciones Complejidad
Peor caso O(n log n) O(n log n)
Mejor caso O(n log n) O(n log n)
Promedio O(n log n) O(n log n)
  • Espacio adicional: O(1) (ordenamiento in-place).
  • Estable: No (puede cambiar el orden relativo de elementos iguales).

Ventajas

  • Muy eficiente para listas grandes.
  • No requiere memoria adicional significativa.

Desventajas

  • No es estable.
  • Más lento que Quick Sort en la práctica para la mayoría de los casos.

Ejemplo en PHP

function heapify(&$arr, $n, $i) {
    $largest = $i;
    $left = 2 * $i + 1;
    $right = 2 * $i + 2;

    if ($left < $n && $arr[$left] > $arr[$largest]) {
        $largest = $left;
    }

    if ($right < $n && $arr[$right] > $arr[$largest]) {
        $largest = $right;
    }

    if ($largest != $i) {
        $temp = $arr[$i];
        $arr[$i] = $arr[$largest];
        $arr[$largest] = $temp;

        heapify($arr, $n, $largest);
    }
}

function heapSort(&$arr) {
    $n = count($arr);

    for ($i = floor($n / 2) - 1; $i >= 0; $i--) {
        heapify($arr, $n, $i);
    }

    for ($i = $n - 1; $i > 0; $i--) {
        $temp = $arr[0];
        $arr[0] = $arr[$i];
        $arr[$i] = $temp;

        heapify($arr, $i, 0);
    }
}
Ejemplo en JavaScript

function heapify(arr, n, i) {
    let largest = i;
    const left = 2 * i + 1;
    const right = 2 * i + 2;

    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }

    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }

    if (largest !== i) {
        [arr[i], arr[largest]] = [arr[largest], arr[i]];
        heapify(arr, n, largest);
    }
}

function heapSort(arr) {
    const n = arr.length;

    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }

    for (let i = n - 1; i > 0; i--) {
        [arr[0], arr[i]] = [arr[i], arr[0]];
        heapify(arr, i, 0);
    }
}

def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2

    if left < n and arr[left] > arr[largest]:
        largest = left

    if right < n and arr[right] > arr[largest]:
        largest = right

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)

    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)
