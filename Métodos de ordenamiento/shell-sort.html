Shell Sort

¿Qué es el Shell Sort?

Shell Sort es un algoritmo de ordenamiento basado en la técnica de inserción directa, pero con una mejora significativa en su rendimiento. Divide el arreglo en subarreglos más pequeños utilizando un "gap" o intervalo, y aplica el ordenamiento por inserción en cada subarreglo. A medida que el gap disminuye, el arreglo se vuelve más ordenado, hasta que el gap es 1 y se realiza un ordenamiento final.

¿Cómo funciona Shell Sort?

  1. Seleccionar un intervalo inicial (gap) basado en una secuencia, como la secuencia de Knuth.
  2. Dividir el arreglo en subarreglos según el intervalo.
  3. Aplicar el ordenamiento por inserción en cada subarreglo.
  4. Reducir el intervalo y repetir el proceso hasta que el gap sea 1.

Pasos del Algoritmo (Detallado)

  1. Elegir un intervalo inicial (gap) basado en una secuencia.
  2. Para cada intervalo:
    • Recorrer los elementos del arreglo separados por el intervalo.
    • Aplicar el ordenamiento por inserción en los elementos separados por el intervalo.
  3. Reducir el intervalo y repetir hasta que sea 1.

Complejidad

Caso Complejidad
Peor caso O(n²)
Mejor caso O(n log n)
Promedio Depende de la secuencia de intervalos
  • Espacio adicional: O(1) (ordenamiento in-place)
  • Estable: No (puede cambiar el orden relativo de elementos iguales)

Ventajas

  • Más eficiente que el ordenamiento por inserción para listas grandes.
  • Fácil de implementar.
  • Requiere poca memoria adicional.

Desventajas

  • El rendimiento depende de la secuencia de intervalos utilizada.
  • No es estable.
Animación: Cómo funciona el Shell Sort
Ejemplo en PHP

function shellSort($arr) {
    $n = count($arr);
    $gap = intdiv($n, 2);

    while ($gap > 0) {
        for ($i = $gap; $i < $n; $i++) {
            $temp = $arr[$i];
            $j = $i;
            while ($j >= $gap && $arr[$j - $gap] > $temp) {
                $arr[$j] = $arr[$j - $gap];
                $j -= $gap;
            }
            $arr[$j] = $temp;
        }
        $gap = intdiv($gap, 2);
    }

    return $arr;
}
Ejemplo en JavaScript

function shellSort(arr) {
    let n = arr.length;
    let gap = Math.floor(n / 2);

    while (gap > 0) {
        for (let i = gap; i < n; i++) {
            let temp = arr[i];
            let j = i;
            while (j >= gap && arr[j - gap] > temp) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            arr[j] = temp;
        }
        gap = Math.floor(gap / 2);
    }

    return arr;
}
Ejemplo en Python

def shell_sort(arr):
    n = len(arr)
    gap = n // 2

    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2

    return arr
