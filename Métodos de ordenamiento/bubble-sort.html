Bubble Sort

¿Qué es el Bubble Sort?

Bubble Sort (Ordenamiento de burbuja) es un algoritmo de ordenamiento simple pero ineficiente para listas grandes. Funciona al recorrer repetidamente la lista, comparando elementos adyacentes e intercambiándolos si están en el orden incorrecto. Al final de cada iteración, el elemento más grande "burbujea" hacia el final de la lista.

¿Cómo funciiona Bubble Sort?

  1. Se recorren los elementos del arreglo de izquierda a derecha.
  2. Si el elemento actual es mayor que el siguiente, se intercambian.
  3. Este proceso se repite hasta que no se requieran más intercambios, lo que significa que el arreglo está ordenado.
  4. Con cada pasada, el número de elementos a comparar se reduce porque los más grandes ya están colocados al final.

Pasos del Algoritmo (Detallado)

  1. Repetir (n - 1) veces:
    • Desde el inicio hasta el último elemento no ordenado:
      • Comparar el elemento actual con el siguiente.
      • Si están en el orden incorrecto, intercambiarlos.
  2. Si no hubo intercambios en una pasada, el arreglo está ordenado.

Complejidad

Caso Comparaciones e intercambios Complejidad
Peor caso Lista invertida O(n²)
Mejor caso Lista ya ordenada O(n)
Promedio Lista aleatoria O(n²)
  • Espacio adicional: O(1) (ordenamiento in-place)
  • Estable: Sí (no cambia el orden relativo de elementos iguales)

Ventajas

  • Fácil de implementar.
  • Intuitivo y útil para fines educativos.
  • No requiere memoria adicional.

Desventajas

  • Muy ineficiente en listas grandes.
  • Existen algoritmos mucho más rápidos como Merge Sort, Quick Sort o Heap Sort.

Ejemplo en PHP

function bubbleSort($arr) {
    $n = count($arr);
    for ($i = 0; $i < $n - 1; $i++) {
        for ($j = 0; $j < $n - $i - 1; $j++) {
            if ($arr[$j] > $arr[$j + 1]) {
                $temp = $arr[$j];
                $arr[$j] = $arr[$j + 1];
                $arr[$j + 1] = $temp;
            }
        }
    }
    return $arr;
}
Ejemplo en JavaScript

function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}
Ejemplo en Python

def bubble_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr
