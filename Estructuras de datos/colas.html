Colas

¿Qué son las colas?

Las colas son estructuras de datos lineales que siguen el principio FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir.

Operaciones Básicas

  • enqueue(elemento) → agregar al final
  • dequeue() → quitar del frente
  • front() → consultar el primero sin quitarlo
  • isEmpty() → verificar si la cola está vacía
  • size() → obtener el tamaño de la cola

Implementaciones Comunes

  • Planificadores de tareas
  • Simulación de procesos
  • Cola de impresión
Animación: Cómo funcionan las colas
Ejemplo en PHP

// Crear una cola
$cola = [];
array_push($cola, "A");
array_push($cola, "B");
array_push($cola, "C");

// Eliminar el primer elemento
$elemento = array_shift($cola);
echo $elemento; // Imprime "A"
Ejemplo en JavaScript

// Crear una cola
const cola = [];
cola.push("A");
cola.push("B");
cola.push("C");

// Eliminar el primer elemento
const elemento = cola.shift();
console.log(elemento); // Imprime "A"
Ejemplo en Python

# Crear una cola
from collections import deque
cola = deque()
cola.append("A")
cola.append("B")
cola.append("C")

# Eliminar el primer elemento
elemento = cola.popleft()
print(elemento) # Imprime "A"
