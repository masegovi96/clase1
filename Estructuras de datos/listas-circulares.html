Listas Circulares

¿Qué son las listas circulares?

Las listas circulares son estructuras de datos donde el último nodo apunta al primer nodo, formando un ciclo continuo.

Operaciones Básicas

  • insert(valor) → agregar al final
  • delete(valor) → eliminar un nodo específico
  • search(valor) → buscar un nodo específico
  • traverse() → se detiene cuando vuelve al inicio

Implementaciones Comunes

  • Juegos por turnos
  • Algoritmos tipo "ronda"
Animación: Cómo funcionan las listas circulares
Ejemplo en PHP

class Nodo {
    public $valor;
    public $siguiente;

    public function __construct($valor) {
        $this->valor = $valor;
        $this->siguiente = null;
    }
}

// Crear nodos
$nodo1 = new Nodo("A");
$nodo2 = new Nodo("B");
$nodo3 = new Nodo("C");

// Enlazar nodos circularmente
$nodo1->siguiente = $nodo2;
$nodo2->siguiente = $nodo3;
$nodo3->siguiente = $nodo1;

// Recorrer la lista circular
$actual = $nodo1;
for ($i = 0; $i < 6; $i++) { // Recorrer 6 nodos para demostrar el ciclo
    echo $actual->valor . " ";
    $actual = $actual->siguiente;
}
Ejemplo en JavaScript

// Clase Nodo
class Nodo {
    constructor(valor) {
        this.valor = valor;
        this.siguiente = null;
    }
}

// Crear nodos
const nodo1 = new Nodo("A");
const nodo2 = new Nodo("B");
const nodo3 = new Nodo("C");

// Enlazar nodos circularmente
nodo1.siguiente = nodo2;
nodo2.siguiente = nodo3;
nodo3.siguiente = nodo1;

// Recorrer la lista circular
let actual = nodo1;
for (let i = 0; i < 6; i++) { // Recorrer 6 nodos para demostrar el ciclo
    console.log(actual.valor);
    actual = actual.siguiente;
}
Ejemplo en Python

# Clase Nodo
class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.siguiente = None

# Crear nodos
nodo1 = Nodo("A")
nodo2 = Nodo("B")
nodo3 = Nodo("C")

# Enlazar nodos circularmente
nodo1.siguiente = nodo2
nodo2.siguiente = nodo3
nodo3.siguiente = nodo1

# Recorrer la lista circular
actual = nodo1
for _ in range(6): # Recorrer 6 nodos para demostrar el ciclo
    print(actual.valor)
    actual = actual.siguiente
