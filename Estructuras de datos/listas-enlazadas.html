Listas Enlazadas

¿Qué son las listas enlazadas?

Las listas enlazadas son estructuras de datos lineales donde cada elemento (nodo) contiene un valor y una referencia al siguiente nodo en la lista.

Operaciones Básicas

  • insertAtBeginning(valor) → insertar al inicio
  • insertAtEnd(valor) → insertar al final
  • insertAt(pos, valor) → insertar en una posición específica
  • deleteAt(pos) → eliminar en una posición específica
  • search(valor) → buscar un valor en la lista
  • traverse() → recorrer elementos

Implementaciones Comunes

  • Implementaciones de otras estructuras
  • Recolección de basura (Garbage Collection)
Animación: Cómo funcionan las listas enlazadas
Ejemplo en PHP

class Nodo {
    public $valor;
    public $siguiente;

    public function __construct($valor) {
        $this->valor = $valor;
        $this->siguiente = null;
    }
}

// Crear nodos
$nodo1 = new Nodo("A");
$nodo2 = new Nodo("B");
$nodo3 = new Nodo("C");

// Enlazar nodos
$nodo1->siguiente = $nodo2;
$nodo2->siguiente = $nodo3;

// Recorrer la lista
$actual = $nodo1;
while ($actual != null) {
    echo $actual->valor . " ";
    $actual = $actual->siguiente;
}
Ejemplo en JavaScript

// Clase Nodo
class Nodo {
    constructor(valor) {
        this.valor = valor;
        this.siguiente = null;
    }
}

// Crear nodos
const nodo1 = new Nodo("A");
const nodo2 = new Nodo("B");
const nodo3 = new Nodo("C");

// Enlazar nodos
nodo1.siguiente = nodo2;
nodo2.siguiente = nodo3;

// Recorrer la lista
let actual = nodo1;
while (actual !== null) {
    console.log(actual.valor);
    actual = actual.siguiente;
}
Ejemplo en Python

# Clase Nodo
class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.siguiente = None

# Crear nodos
nodo1 = Nodo("A")
nodo2 = Nodo("B")
nodo3 = Nodo("C")

# Enlazar nodos
nodo1.siguiente = nodo2
nodo2.siguiente = nodo3

# Recorrer la lista
actual = nodo1
while actual is not None:
    print(actual.valor)
    actual = actual.siguiente
