Pilas

¿Qué son las pilas?

Las pilas son estructuras de datos lineales que siguen el principio LIFO (Last In, First Out), donde el último elemento en entrar es el primero en salir.

Operaciones Básicas

  • push(elemento) → inserta elemento arriba
  • pop() → elimina el último elemento
  • peek() o top() → consulta el elemento superior
  • isEmpty() → verifica si está vacía
  • size() → retorna el tamaño de la pila

Implementaciones Comunes

  • Recursión
  • Reversión de cadenas
  • Deshacer acciones
Animación: Cómo funcionan las pilas
Ejemplo en PHP

// Crear una pila
$pila = [];
array_push($pila, "A");
array_push($pila, "B");
array_push($pila, "C");

// Eliminar el último elemento
$elemento = array_pop($pila);
echo $elemento; // Imprime "C"
Ejemplo en JavaScript

// Crear una pila
const pila = [];
pila.push("A");
pila.push("B");
pila.push("C");

// Eliminar el último elemento
const elemento = pila.pop();
console.log(elemento); // Imprime "C"
Ejemplo en Python

# Crear una pila
pila = []
pila.append("A")
pila.append("B")
pila.append("C")

# Eliminar el último elemento
elemento = pila.pop()
print(elemento) # Imprime "C"
