Grafos

¿Qué son los grafos?

Un grafo es una estructura de datos que consiste en un conjunto de nodos (o vértices) conectados por aristas. Los grafos pueden ser dirigidos o no dirigidos, y pueden tener pesos asociados a las aristas.

Operaciones Básicas

  • addVertex(v) → agregar un vértice al grafo
  • addEdge(v1, v2) → agregar una arista entre dos
  • removeEdge(v1, v2) → eliminar una arista entre dos
  • removeVertex(v) → eliminar
  • adjacent(v1, v2) → verificar si hay una arista entre dos
  • Recorridos: BFS (amplitud), DFS (profundidad)
  • shortestPath(v1, v2) (usando Dijkstra, etc.)

Tipos de recorridos

  1. Búsqueda en profundidad (DFS - Depth First Search)
    • Explora lo más lejos posible antes de retroceder.
    • Usa una pila (implícita en la recursión) o una pila explícita.
    • Ideal para detectar ciclos, caminos, componentes conexos.
  2. Búsqueda en anchura (BFS - Breadth First Search)
    • Explora todos los vecinos antes de ir más profundo.
    • Usa una cola.
    • Útil para encontrar el camino más corto en grafos no ponderados.

Principales algoritmos

  1. Dijkstra
    • Solo funciona con pesos positivos.
    • Usa una cola de prioridad para elegir el nodo con menor costo actual.
    • Ideal para: grafos ponderados sin pesos negativos.
  2. Bellman-Ford
    • Permite aristas con pesos negativos.
    • Detecta ciclos negativos.
    • Más lento que Dijkstra.
    • Ideal para: redes con deudas, economía, grafos dirigidos con ciclos.
  3. Floyd-Warshall
    • Calcula el camino más corto entre todos los pares de nodos.
    • Usa una matriz de adyacencia.
    • Ideal para: grafos pequeños y completamente conectados.
  4. A* (A-star)
    • Usa heurísticas para estimar la distancia restante.
    • Muy usado en videojuegos e inteligencia artificial.
    • Ideal para: navegación, mapas y sistemas en tiempo real.

Implementaciones Comunes

  • Rutas GPS
  • Redes sociales
  • Circuitos

Ejemplo en PHP

class Grafo {
    private $adyacencia = [];

    public function agregarArista($origen, $destino) {
        $this->adyacencia[$origen][] = $destino;
    }

    public function mostrarGrafo() {
        foreach ($this->adyacencia as $origen => $destinos) {
            echo "$origen -> " . implode(", ", $destinos) . PHP_EOL;
        }
    }
}

// Crear grafo
$grafo = new Grafo();
$grafo->agregarArista("A", "B");
$grafo->agregarArista("A", "C");
$grafo->agregarArista("B", "D");
$grafo->agregarArista("C", "D");
$grafo->mostrarGrafo();
Ejemplo en JavaScript

// Clase Grafo
class Grafo {
    constructor() {
        this.adyacencia = {};
    }

    agregarArista(origen, destino) {
        if (!this.adyacencia[origen]) this.adyacencia[origen] = [];
        this.adyacencia[origen].push(destino);
    }

    mostrarGrafo() {
        for (const origen in this.adyacencia) {
            console.log(`${origen} -> ${this.adyacencia[origen].join(", ")}`);
        }
    }
}

// Crear grafo
const grafo = new Grafo();
grafo.agregarArista("A", "B");
grafo.agregarArista("A", "C");
grafo.agregarArista("B", "D");
grafo.agregarArista("C", "D");
grafo.mostrarGrafo();
Ejemplo en Python

class Grafo:
    def __init__(self):
        self.adyacencia = {}

    def agregar_arista(self, origen, destino):
        if origen not in self.adyacencia:
            self.adyacencia[origen] = []
        self.adyacencia[origen].append(destino)

    def mostrar_grafo(self):
        for origen, destinos in self.adyacencia.items():
            print(f"{origen} -> {', '.join(destinos)}")

# Crear grafo
grafo = Grafo()
grafo.agregar_arista("A", "B")
grafo.agregar_arista("A", "C")
grafo.agregar_arista("B", "D")
grafo.agregar_arista("C", "D")
grafo.mostrar_grafo()
